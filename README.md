# 📈 OptimizAR Digital — Marketing Studio Website

A modern, responsive landing page for a digital marketing studio.  
Built with pure HTML, CSS, and JavaScript - no frameworks, no backend, just fast and polished frontend.

---

## 📌 Description

**OptimizAR Digital** is a presentation website for a digital marketing studio. It showcases the agency's services, portfolio, team, and contact information in a clean, professional layout designed to convert visitors into clients.

---

## 🧠 Features

- 🏠 Hero section with studio introduction and call-to-action
- 📋 Services section (SEO, SMM, ads, branding, etc.)
- 💼 Portfolio / case studies showcase
- 👥 Team presentation
- 📞 Contact section with form or info
- ⚡ Smooth scrolling and JS-powered interactions
- 📱 Fully responsive — mobile, tablet, and desktop
- 🎨 Custom CSS design with modern aesthetics

---

## 🛠 Technologies Used

| Layer       | Technology       |
|-------------|------------------|
| Markup      | HTML5            |
| Styling     | CSS3             |
| Interactivity | Vanilla JavaScript |

## 🚀 Quick Start

### Clone the Repository
```bash
git clone https://github.com/babayaga0x/OptimizAR_Digital.git
cd OptimizAR_Digital
```

### Open in Browser

Just open `index.html` directly:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or use a local dev server for a better experience:
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

Then visit `http://localhost:8080`.

## 🎨 Design Highlights

- **Single-page layout** — all content flows on one scrollable page
- **Sticky / animated navbar** — highlights active section on scroll
- **Smooth scroll navigation** — anchor links with smooth behavior
- **Mobile hamburger menu** — responsive nav for small screens
- **Hover effects and transitions** — polished micro-interactions via CSS
